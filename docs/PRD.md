# Product Requirements Document (PRD)
## 项目：诺曼底登陆 (D-Day Defense)

---

## 1. 背景与目标

开发一款"诺曼底登陆"主题的塔防网页游戏，为年轻人提供简单有趣的休闲游戏体验。玩家通过在黑色画布上布置防御设施，抵御不断来袭的敌军士兵和登陆艇。

## 2. 目标用户

喜欢玩网页休闲小游戏的年轻玩家，追求快节奏、易上手的游戏体验。

## 3. 核心功能

| 功能 | 描述 |
|------|------|
| **放置防御塔** | 在画布网格上点击放置防御塔，自动攻击范围内的敌人 |
| **模式选择** | 游戏开始前选择"经典模式"（固定波次）或"无尽模式"（无限波次） |
| **游戏结算** | 游戏结束时显示：达到等级、击杀数、坚持波次 |

## 4. 非目标

以下功能**不在此版本范围内**：

- 用户注册/登录系统
- 排行榜/在线对战
- 复杂的防御塔升级树（只保留简单升级）
- 存档系统（刷新页面重新开始）
- 移动端适配（专注桌面体验）

## 5. 完成标准

项目可验收的标准：

- [ ] 画布正确渲染黑色背景和游戏元素（防御塔、敌人、登陆艇）
- [ ] 能够放置至少2种不同类型的防御塔
- [ ] 敌人沿固定路径移动，防御塔可攻击并消灭敌人
- [ ] 经典模式至少包含5波敌人
- [ ] 游戏结束正确显示三项统计数据
- [ ] 无JavaScript运行时错误

## 6. 风险与降级方案

| 风险 | 降级方案 |
|------|----------|
| 敌人AI/路径寻找太复杂 | 改为直线移动，不做复杂寻路 |
| 两种模式都来不及做 | 只做经典模式，砍掉无尽模式 |
| 防御塔升级太复杂 | 移除升级功能，只保留放置 |
| 登陆艇动画太难实现 | 用简单几何图形代替，或只做士兵 |
